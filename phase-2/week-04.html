SELECT, WHERE, ORDER BY, LIMIT, basic aggregates. Finish Kaggle Intro to SQL for the cert. Build the confidence to query a database you have never seen before.

Learning Goals

  • Drill Skill #2 in a new context — SDQS loop: Schema, Data question, Query, Validate, Stress-test

Content Releases After Week 3

Full SDQS loop pattern, 10 Chinook questions, sandbox walkthrough, and portfolio template unlock after Phase 1 completion.

AI Co-Work Pattern (Preview) — The SDQS Loop

Five letters: Schema → Data question → Query → Validate → Stress-test. You paste the schema every session, state the question precisely, let Claude write the SQL, run it in a sandbox, then make Claude prove the numbers.

Resources (Preview)

  • SQLBolt — browser-interactive, free, no signup
  • Kaggle Intro to SQL (free cert)
  • sqliteonline.com — primary sandbox
  • DB Fiddle — fallback sandbox
  • Chinook sample DB
  • Alex the Analyst: SQL playlist (see Alex Companion)

Mini-Project (Preview)

Using Chinook, answer ten analyst questions in SQL — top customers by spend, monthly revenue trend, median invoice size, month-over-month growth with flags, and more. Every query includes the SQL, a plain-English description, and a short finance-lens interpretation.
