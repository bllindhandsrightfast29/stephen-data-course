Phase 2 — SQL

Weeks 4–6Certs: Kaggle Intro SQL + Advanced SQL

Three weeks on SQL. Weeks build strictly: basics, then joins, then analyst-grade queries with CTEs and window functions. Chinook and Northwind as practice schemas. By end of phase you can query a database you have never seen before.

Phase Goal

Fluency in SQL to the point that Claude writes the query, you read it critically, and you catch the bugs. You will do reps across two databases so the schema becomes background noise and SQL becomes the foreground skill.

Weeks in This Phase
