Week 6 — Analyst-Grade SQL: CTEs, Window Functions, Subqueries

Phase 2 · SQLSkill Focus: Iterate with AI~15 hours
Course Progress
Week 6 of 12

CTEs for readability. Window functions for ranks, running totals, and moving averages. This is where analyst-grade SQL lives. Build the Treasury Forecast Query — an interview screenshot you can walk through in five minutes.

Learning Goals

  • WITH clauses (CTEs), ROW_NUMBER, RANK, LAG, LEAD, moving averages
  • Drill Skill #4: iterate with AI through 4 rounds to refine a multi-step analysis
  • Ship: Treasury Forecast Query + Kaggle Advanced SQL cert

Content Releases After Week 5

Full iteration-loop pattern, forecast-query scaffold, and walkthrough template unlock after Week 5 commit.

AI Co-Work Pattern (Preview)

Skill #4: your first prompt is almost never the right one, and that is OK. Iteration is the skill. Build a 5-CTE query through 4 rounds of refinement. Reps teach you to patch, not rewrite — accept 70%-right answers and push Claude to fix the last 30%.

Resources (Preview)

  • Kaggle Advanced SQL (free cert)
  • Mode Analytics SQL tutorials (free, advanced)
  • DataTalksClub zoomcamp (reference, not redistribution)
  • Window functions visual guide (Lukas Eder posts)

Mini-Project (Preview) — The Treasury Forecast Query

A single readable SQL query using CTEs that returns, per customer: total lifetime spend, spend per quarter (last 4), 4Q moving average, QoQ growth rate, country rank by spend, and a flag when recent spend is less than 50% of the 4Q MA. A strong portfolio and interview-walkthrough artifact.
