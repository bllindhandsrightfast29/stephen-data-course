Week 10 — Python & Pandas Foundations

Phase 4 · PythonSkill Focus: Command AI (pandas)~15 hours
Course Progress
Week 10 of 12

Python syntax. Pandas DataFrame fundamentals. Google Colab as your working environment. Rebuild Chinook in pandas so you can translate fluently between SQL and pandas — most of what pandas interviews test.

Learning Goals

  • Python syntax basics (variables, control flow, functions, comprehensions)
  • Pandas fundamentals: read_csv, filter, groupby, merge
  • Drill Skill #2 again in pandas context — LDTV loop: Load, Describe, Transform, Validate
  • Ship: Chinook-in-pandas notebook + reflection on SQL-vs-pandas tradeoffs

Content Releases After Week 9

Full LDTV loop drill, Colab setup walkthrough, and Chinook-in-pandas rebuild template unlock after Phase 3 completion.

AI Co-Work Pattern (Preview) — The LDTV Loop

Load (paste df.info() and df.head(), confirm Claude sees the dtypes). Describe (English transformation, not code). Transform (Claude writes pandas, you run in Colab). Validate (paste output, ask Claude to walk the cross-check vs an equivalent SQL path).

Resources (Preview)

  • Kaggle Python (skip if Python is familiar)
  • Kaggle Pandas (free cert, core this week)
  • Google Colab (free T4 tier)
  • MS DS-for-Beginners Python section
  • Alex the Analyst: Python & pandas videos (see Alex Companion)

Mini-Project (Preview)

Redo the ten Chinook SQL questions from Week 4 in pandas. Same answers, different tool. Add a reflection: “Which tool was better for each question and why?” The reflection is a useful interview-talking-point in itself.

Cert Progress

Kaggle Pandas cert earned. Fifth (or fourth, if GAIQ skipped) free cert in the portfolio.
