Week 2 — Advanced Sheets & Excel (AI-Assisted)

Phase 1 · FoundationSkill Focus: Command AI to Execute~14 hours
Course Progress
Week 2 of 12

Advanced Excel/Sheets mechanics. Build muscle memory for “describe what I want, get a formula, validate it.” By week's end you can run a full Treasury Ops Dashboard workbook without writing a formula from scratch.

Learning Goals

  • Master INDEX/MATCH, XLOOKUP, dynamic arrays, LAMBDA, pivots with calculated fields
  • Drill Skill #2: command AI to execute a data question — formula edition
  • Ship the Treasury Operations Dashboard Workbook to your portfolio

Content Releases After Week 1

Full drill content, mini-project details, prompt library, and portfolio template will unlock after you complete and commit the Week 1 portfolio entry. Preview the shape of the week below so you can plan.

AI Co-Work Pattern (Preview)

Skill #2: You describe a transformation in plain English, Claude returns a formula, you paste into Sheets, you validate. Sample drill:

I have a Google Sheet with columns A=invoice_date, B=customer,
C=amount, D=paid_date. I want a formula in column E that returns
"days to pay" when D is populated, "open - [N] days" when D is
blank and A is more than 30 days old, and "current" otherwise.
Write me the formula.

That is one Skill #2 rep. You will do twenty this week.

Resources (Preview)

  • Microsoft Excel/Sheets advanced tutorials (YouTube: ExcelIsFun, free)
  • Google Sheets Advanced Formulas cheat sheet
  • SEC EDGAR financial statement data sets (public domain)
  • FRED for macro context (free API key)
  • Alex the Analyst: Excel playlist (see Alex Companion)

Mini-Project (Preview)

Build a Treasury Operations Dashboard Workbook with five tabs: raw AP/AR data (500 rows, realistic), aged receivables pivot, 12-week cash flow forecast, variance analysis, and written executive summary. Every formula gets a comment explaining what it does. Every calculation is Claude-assisted, then manually verified against a second method.

Cert Progress

CFI Excel Crash Course (free tier) is the target this week. Optional but a clean resume line.
