# Stephen Data Pivot — Course Site

This repository hosts the **Claude Co-Work: Data Analytics Career Pivot** course for Stephen Branch. The course is delivered as a static GitHub Pages site — not a collection of markdown files to read in GitHub.

## Visit the Course Site

Once GitHub Pages is enabled (Settings → Pages → Deploy from branch → `master` root), the course lives at:

`https://bllindhandsrightfast29.github.io/stephen-data-pivot/`

Start at the course home, then click into Week 1 to begin.

## Structure

- `index.html` — course syllabus / home page
- `phase-1/` through `phase-4/` — four phases, each with an overview and three week pages
- `cert-tracker.html` — five certifications tracked across the 12 weeks
- `resources/` — prompt library, dataset inventory, job search kit, Alex the Analyst companion
- `assets/css/main.css` — LMS-style theme (inspired by HTML5 UP Editorial, CC-BY 3.0)

## Setup

No build step. Plain HTML and CSS. GitHub Pages serves the files directly. The `.nojekyll` file in the repo root tells Pages to skip Jekyll processing.
