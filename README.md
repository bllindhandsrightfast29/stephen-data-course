# Stephen Data Course

Applied Data Analytics — a 12-week course for Stephen Branch. Delivered as a static GitHub Pages site, not a collection of markdown files.

## Visit the Course Site

Once GitHub Pages is enabled (Settings → Pages → Deploy from branch → `main` root), the course lives at:

`https://bllindhandsrightfast29.github.io/stephen-data-course/`

Start at the course home, then click into Week 1 to begin.

## Structure

- `index.html` — course home
- `phase-1/` through `phase-4/` — four phases, each with an overview and three week pages
- `cert-tracker.html` — five certifications tracked across the 12 weeks
- `resources/` — prompt library, dataset inventory, job search kit, Alex the Analyst companion
- `assets/css/main.css` — LMS-style theme (typography inspired by editorial web layouts)

## Setup

No build step. Plain HTML and CSS. GitHub Pages serves the files directly. The `.nojekyll` file in the repo root tells Pages to skip Jekyll processing.
