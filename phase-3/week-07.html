Week 7 — Dashboard Fundamentals with Tableau Public

Phase 3 · VisualizationSkill Focus: Present the Insight~15 hours
Course Progress
Week 7 of 12

Chart type literacy. Tableau Public end-to-end: connect, build, publish. Non-terrible dashboard design principles (Few, Knaflic). Build the Macro Weather Report using FRED data.

Learning Goals

  • When to use bar vs line vs scatter vs area vs heatmap (the right chart is rarely the first one)
  • Drill Skill #5: present the insight to non-technical stakeholders (viz edition)
  • Ship: FRED Macro Weather Report, published to Tableau Public with a shareable link

Content Releases After Week 6

Full Ten-Second Test pattern, FRED data pull walkthrough, and critique-rounds template unlock after Phase 2 completion.

AI Co-Work Pattern (Preview) — The Ten-Second Test

Build a draft, screenshot it, paste to Claude, ask for critique on four dimensions: 10-second insight, cognitive load, chart-type choices, story arc. Claude gives direct feedback. You iterate. Stop when Claude says “this lands.”

Resources (Preview)

  • Tableau Public — free, workbooks downloadable
  • Tableau training videos (free tier)
  • Kaggle Data Visualization (free cert)
  • Cole Knaflic “Storytelling With Data” blog
  • FRED — primary data source this week
  • Alex the Analyst: Tableau playlist (see Alex Companion)

Mini-Project (Preview) — Macro Weather Report

Using FRED: US unemployment (5yr trend), CPI with Fed target overlay, 10Y–2Y Treasury spread with recession flag, GDP growth YoY bars, plus one executive-summary text box with your interpretation. Publish to Tableau Public. Share the link.
